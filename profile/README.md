### 핵심 기능

**음성 분석 & 교정**
- OpenVoice 기반의 실시간 발음 분석 및 맞춤형 피드백 제공

**AI 대화 학습**  
- ChatGPT API로 자연스러운 한국어 회화 연습 지원

**발음 평가 시스템**
- MFCC + DTW 알고리즘을 활용한 정밀한 발음 점수화

**입 모양 분석**
- MediaPipe + MobileNet/LSTM으로 시각적 발음 피드백 제공
  
**롤플레이 학습**
- 친구와 함께하는 상황극 기반 대화 시뮬레이션
